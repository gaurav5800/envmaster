# 🚀 EnvMaster

A modern CLI to validate, generate and manage `.env` files.

## ✨ Features

- 🔍 Validate environment files
- 🔄 Sync `.env.example` with `.env`
- 📦 Detect missing variables
- ⚡ Fast TypeScript CLI
- 🧪 Unit tested with Vitest
- 🚀 GitHub Actions CI

## 📦 Installation

```bash
npm install -g envmaster
```

## 🚀 Usage

### Check your environment

```bash
envmaster doctor
```

### Sync `.env.example`

```bash
envmaster sync
```
